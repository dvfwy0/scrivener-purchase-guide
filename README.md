## 几个让我觉得"哎有点东西"的功能

**软木板（Corkboard）**：把每个章节、每个场景做成一张虚拟索引卡，钉在软木板上，拖一拖就能调整结构。比你在墙上贴满便利贴然后撕了重贴方便一百倍。

**资料管理（Research）**：PDF、图片、网页、采访录音，全部塞进项目里，写作时随时调出来对照。再也不用 Alt+Tab 在十几个窗口之间手忙脚乱。

**分屏编辑**：左边写正文，右边放参考资料或前一章节，对照着写不费劲。

**全屏写作模式**：把所有工具栏、菜单、通知都藏起来，只剩你和文字。专注才是王道。

**编译导出**：写完一键编译成 Word、PDF、Final Draft、纯文本、ePub 等格式，甚至能用不同字体写作、用投稿要求的格式导出。
